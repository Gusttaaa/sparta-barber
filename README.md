# Sparta Barber

Plataforma web para gerenciamento e agendamento de uma barbearia com design espartano, desenvolvida com Next.js.

## Funcionalidades

- 📅 Sistema de agendamento online
- 👤 Área do cliente com histórico de agendamentos
- 🛍️ Loja integrada de produtos
- 👨‍💼 Admin dashboard para gerenciar agendamentos e serviços
- 📱 Design responsivo mobile-first
- ⚔️ Design temático espartano

## Stack

- **Frontend**: Next.js 15, React, TypeScript
- **Styling**: Tailwind CSS, Shadcn/ui
- **Backend**: Next.js API Routes
- **Database**: Supabase (PostgreSQL)
- **Deployment**: Vercel

## Setup Local

```bash
# Instalar dependências
npm install

# Configurar variáveis de ambiente
# Criar arquivo .env.local com credenciais Supabase

# Rodar servidor de desenvolvimento
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000)

## Deploy

Deploy automático via Vercel. Conecte seu repositório GitHub para CI/CD contínuo.

## Estrutura

```
app/              # Rotas e páginas
components/       # Componentes React
lib/              # Utilitários e configurações
public/           # Arquivos estáticos
```
